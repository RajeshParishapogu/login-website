#Flask Login System
## Description
This project is a simple login and registration system using Python framework-Flask & ML Algorithms , Fronted-HTML,CSS.
It allows users to create accounts and log in securely.

##Features
-User Registration
-User Login

##Tech Stack
-Python (Flask)
-HTML
-CSS
-ML Alogithms.

##How to Run
1. Install  Flask
2. Run app.py
3. Open browser and go to localhost

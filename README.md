## Flask Login System


## Description
This project is a login and registration system build using Flask.
It allows users to create accounts and log in securely log in.

## Features
- User Registration
- User Login Autentication

##Tech Stack
- Python (Flask)
- HTML
- CSS
- ML Alogithms (if implemented)

## Machine Learning Usage
This project uses basic  pattern-based logic to validate user inputs and improve authentication behavior.

1. Install  Flask
2. Run : app.py
3. Open browser and go to localhost
